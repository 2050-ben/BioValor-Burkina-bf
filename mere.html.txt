            </div>
            
            <div class="section-content">
                <h3><i class="fas fa-book-open"></i> Contexte et Enjeux</h3>
                <p>L'agriculture représente ~33 % du PIB et emploie plus de 70 % de la population active burkinabè, majoritairement rurale. C'est une agriculture familiale diversifiée (mil, sorgho, maïs, niébé, arachide, etc.) souvent pratiquée de manière traditionnelle.</p>
                <p>Les pratiques ancestrales (techniques Zai, demi-lunes, cordons pierreux) sont mises en avant pour restaurer la fertilité et capter l'eau. La cohabitation élevage/cultures est fréquente pour valoriser les pâturages.</p>
                <p><strong>Objectifs :</strong> Assurer la sécurité alimentaire nationale, augmenter les revenus agricoles, moderniser sans détruire l'environnement.</p>
            </div>

            <div class="section-content">
                <h3><i class="fas fa-exclamation-triangle"></i> Défis et Solutions</h3>
                <div class="defis-astuces">
                    <div class="defis">
                        <h3><i class="fas fa-times-circle"></i> Défis</h3>
                        <p><strong>Dégradation des terres :</strong> Épuisement continu des sols, variabilité climatique avec sécheresses récurrentes.</p>
                        <p><strong>Ressources limitées :</strong> Accès limité à l'eau et aux semences améliorées.</p>
                        <p><strong>Infrastructures :</strong> Manque d'infrastructures (irrigation, stockage, routes rurales) et de capitaux.</p>
                        <p><strong>Organisation :</strong> Dispersion des petits exploitants, faible formation technique.</p>
                    </div>
                    <div class="astuce">
                        <h3><i class="fas fa-lightbulb"></i> Solutions Proposées</h3>
                        <p><strong>Coopératives agricoles :</strong> Mutualiser outils et achats pour réduire les coûts.</p>
                        <p><strong>Microcrédits :</strong> Financement ciblé pour irrigation solaire et équipements.</p>
                        <p><strong>Formation :</strong> Centres de formation agricole et extension rurale renforcée.</p>
                        <p><strong>Politique de prix :</strong> Prix d'achat minimum pour soutenir les cultivateurs.</p>
                    </div>
                </div>
            </div>

            <div class="section-content">

                            <p>Étudier l'ajout d'amendements (compost, biochar local) dans les trous de Zai pour maximiser l'infiltration d'eau et la nutrition des plantes.</p>

                            <p>Recherche sur des combinaisons de cultures vivrières et d'arbres (baobab, jujubier) pour retenir l'eau et enrichir les sols.</p>
                            <span class="research-ref">Réf : ICRAF, 2024</span>

                        <p><strong>Infrastructure :</strong> Mauvaise conservation post-récolte, routes impraticables en saison des pluies.</p>
                        <p><strong>Accès aux marchés :</strong> Difficulté d'accès aux marchés urbains pour les surplus ruraux.</p>
                        <p><strong>Nutrition :</strong> Dépendance au sorgho/millet; manque de protéines et micronutriments.</p>

                        <p><strong>Aides alimentaires :</strong> Systèmes de "bons alimentaires" en période de soudure.</p>
                        <p><strong>Diversification :</strong> Renforcer pisciculture et aviculture villageoise.</p>

        // Navigation fluide
        document.querySelectorAll('a[href^="#"]').forEach(anchor => {
            anchor.addEventListener('click', function(e) {
                e.preventDefault();
                
                const target = document.querySelector(this.getAttribute('href'));
                if (target) {
                    window.scrollTo({
                        top: target.offsetTop - 80,
                        behavior: 'smooth'
                    });
                    
                    // Mise à jour de la navigation active
                    document.querySelectorAll('nav ul li a').forEach(link => {
                        link.classList.remove('active');
                    });
                    this.classList.add('active');
                }
            });
        });

        // Gestion du formulaire de contact
        const contactForm = document.getElementById('contactForm');
        if (contactForm) {
            contactForm.addEventListener('submit', function(e) {
                e.preventDefault();
                
                // Affichage du message de succès
                const successMessage = document.getElementById('successMessage');
                if (successMessage) {
                    successMessage.style.display = 'block';
                }
                
                // Envoi réel du formulaire
                const formData = new FormData(this);
                
                // Utilisation de Formspree pour l'envoi des emails
                fetch('https://formspree.io/f/mvojrzkq', {
                    method: 'POST',
                    body: formData,
                    headers: {
                        'Accept': 'application/json'
                    }
                })
                .then(response => {
                    if (response.ok) {
                        // Réinitialisation du formulaire
                        contactForm.reset();
                    } else {
                        alert("Une erreur s'est produite. Veuillez réessayer.");
                    }
                })
                .catch(error => {
                    console.error('Erreur:', error);
                });
            });
        }

        // Animation des éléments au défilement
        const observer = new IntersectionObserver((entries) => {
            entries.forEach(entry => {
                if (entry.isIntersecting) {
                    entry.target.classList.add('visible');
                }
            });
        }, {
            threshold: 0.1
        });
        
        document.querySelectorAll('.section-content, .research-card, .tip-card, .blog-post').forEach(el => {
            el.classList.add('fade-in');
            observer.observe(el);
        });
        
        // Ajout du style pour les animations
        const style = document.createElement('style');
        style.textContent = `
            .fade-in {
                opacity: 0;
                transform: translateY(30px);
                transition: opacity 0.6s ease-out, transform 0.6s ease-out;
            }
            
            .fade-in.visible {
                opacity: 1;
                transform: translateY(0);
            }
        `;
        document.head.appendChild(style);
    </script>
